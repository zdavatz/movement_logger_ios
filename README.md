# movement_logger_ios

Movement Logger GUI for iOS — SwiftUI + CoreBluetooth client for the PumpTsueri SensorTile.box. Ported from [movement_logger_android](https://github.com/zdavatz/movement_logger_android).

Open `MovementLogger.xcodeproj` in Xcode 15+, set your signing team on the `MovementLogger` target, and run on a physical iOS 17+ device (BLE doesn't work in the Simulator).
